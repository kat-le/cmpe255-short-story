# cmpe255-short-story

This repository contains materials related to the research paper on a new machine learning approach for clustering ALS patients based on their ALSFRS-R progression. The paper explains how weak supervision, descriptive feature extraction, and distance learning can create clinically meaningful patient subtypes. 

Slides, video presentation, and Medium article are included in INDEX.md

Link to paper: https://arxiv.org/pdf/2511.01945
